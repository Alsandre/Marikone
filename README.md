_To run on local server:_
1. Open **terminal** (<kbd>ctrl</kbd>+<kbd>shift</kbd>+<kbd>`</kbd> or terminal>new terminal)
2. Run comand ```yarn```
3. then ```yarn dev``` 
4. Page will load on ```localhost:3000```
